
README (ESPAÑOL)
---------------

Descripción del Proyecto:
Este proyecto analiza diversos aspectos de los datos financieros de Bitcoin, utilizando técnicas estadísticas y de modelado predictivo para extraer insights valiosos que ayuden a entender su comportamiento en el mercado.

Instrucciones de Ejecución:
- Asegúrate de tener Python y las bibliotecas necesarias instaladas (pandas, matplotlib, statsmodels).
- Ejecuta cada script en orden para replicar los análisis.
- Los scripts están diseñados para ser autocontenidos y fáciles de seguir.

Cómo Contribuir:
- Para contribuir al proyecto, puedes clonar el repositorio y enviar pull requests con tus propias mejoras o correcciones.

Contacto:
- Para más información o preguntas, puedes contactar al desarrollador a través de [bonifazdaniel@gmail.com].
